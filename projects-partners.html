현재 진행중인 프로젝트

올아이티는 다양한 산업 현장에서 MES, ERP, 웹 시스템 구축 프로젝트를 진행하고 있습니다. 제조·인쇄·패키징 분야의 고객사와 함께, 데이터 중심의 스마트 생산환경을 구현하고 있습니다.

지금도 여러 기업과의 협력을 통해 시스템 통합, 클라우드 전환, 공정 자동화 등 실제 현장의 문제를 해결하는 프로젝트가 활발히 진행 중입니다.

HPM MES 시스템 고도화

생산실적, 자재 투입, 품질검사 데이터를 통합 관리하는 MES 시스템을 고도화 개발 중입니다. 공정별 실시간 모니터링과 Lot 추적 기능을 통해 불량률 감소와 생산 효율 향상을 목표로 합니다.

.NET 8 SQL Server DevExpress WinForms

React TypeScript REST API Nginx

우리의 협력사

올아이티는 단순한 시스템 납품이 아니라, 구축 이후의 지속적인 운영과 품질 개선을 중요하게 생각합니다. 이를 위해 각 분야의 전문 파트너들과 긴밀하게 협력하며 고객사의 안정적인 서비스를 지원하고 있습니다.

시스템 유지보수, 업무 컨설팅, 클라우드 인프라, UI/UX 개선 등 프로젝트 이후에도 함께 성장하는 신뢰 기반의 파트너십을 이어가고 있습니다.
